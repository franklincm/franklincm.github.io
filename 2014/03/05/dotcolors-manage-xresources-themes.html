dotcolors - manage Xresources themes

05 Mar 2014 Author: chase

Last night I was looking at a simple litte tool I made to make changing my terminal themes easier called dotcolors, and decided to finally knock out a couple items on the TODO list for it.

dotcolors isn't anything special but it is handy it you're addicted to urxvt and tweaking appearance settings like I am. Simply put, it downloads .Xresources configs from dotshare.it and provides a simple menu to change and apply those settings easily.

Usage

It already had some command line switches to get themes, or change how many themes to display per page etc:

Menu

And the menu is essentially the same except now it can set my transparency, as well as attempts to auto format (read: strip junk from) the config files it downloads. Also you can set a prefix for your ~/.Xresources, the default is urxvt but if you insist on using eterm or what have you this should work. Otherwise you'll just have to edit you're configs the old fashioned way.
