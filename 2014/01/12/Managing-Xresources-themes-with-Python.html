Managing Xresources themes with Python

12 Jan 2014 Author: chase

So I had this simple python script lying around that I used to automate changing color themes in urxvt. It wasn't until I gave it to a friend recently that I felt compelled to actually make it presentable. (read: obsess for 4 days, rewrite entire script + a few hunder additional lines)

Here is the result: github.com/gnullbyte/dotcolors

It now fetches all available Xresources themes from dotshare.it and presents a simple menu to load a new one. I think I might look into pygtk/cairo to see if I can get a live preview of the themes while browsing working.

I use .Xresources to setup everything else that isn't color related. I let dotcolors keep track of the themes in ~/.config/dotcolors/ and I use .Xresources for persistent settings (fonts, scrollbars, perl-ext's, etc).
