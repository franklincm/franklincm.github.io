Android Weechat

11 May 2014 Author: chase

** I need to update this. I no longer user SSHTunnel in favor of OpenVPN **

Weechat Android allows you to connect to a running Weechat instance via a relay from your android device. I've been using it more and more lately. Source Here

  1. ssh server running and working correctly

  2. put keys on phone i.e. /sdcard/sshtunnel/key/id_rsa

  3. configure weechat relay, use whatever port, 6771 is arbitrary

    bind_address restricts connections to localhost only (tunnel)

  4. install/configure SSHTunnel on phone

    • host, port, user
    • password/passphrase (passphrase for unlocking key)
    • do not use socky proxy (leave unchecked)
    • local port i.e. 1234
    • remote address = 127.0.0.1
    • remote port = 6771
    • auto connect, auto reconnect, notfication settings etc.

  5. configure Weechat Android

    • relay host = localhost
    • relay port = 6771
    • relay password = "password" (from step 3, if set)
    • connect automatically
    • connection type = plain connection
    • do not configure tunnel setting within Weechat Android

  6. Start SSHTunnel, then start Weechat Android

  7. Cross fingers!
