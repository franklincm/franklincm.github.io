Installing Grub on UEFI

03 Dec 2013 Author: chase

I've done this enough times I should have it committed to memory.

  1. Create a 512M GPT partition with type ef00 ($esp)

  2. Format as fat32

  3. Install grub

  4. Mount $esp at /boot/efi

  5. Mount efivars if not already mounted

  6. Install grub
